# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-09

### Added
- Initial release
- Hero section with animated CMYK ring, floating tags and stats
- Services grid — 6 typed service cards with Lucide icons
- Process section — 4-step workflow with decorative gradient line
- Portfolio gallery — asymmetric 12-column grid
- Testimonials section — 3 review cards with dynamic star rating
- FAQ accordion — accessible, keyboard navigable
- CTA band with WhatsApp and phone CTAs
- Footer with social links, navigation and contact info
- WhatsApp FAB with pre-filled quote message
- CMYK marquee with fade edges
- Mobile navigation with Escape key support
- Security headers: CSP, X-Frame-Options, HSTS, nosniff, Referrer-Policy, Permissions-Policy
- Typed data layer in `lib/constants.ts`
- Skip link for WCAG accessibility
- `noopener noreferrer` on all external links
